# npm 安装说明

npm（Node Package Manager）是 Node.js 的包管理器，通常随 Node.js 一起安装。

## 安装方法

### 方法 1：安装 Node.js（推荐）

npm 通常随 Node.js 一起安装，所以您需要先安装 Node.js：

1. **访问 Node.js 官网**
   - 打开浏览器访问：https://nodejs.org/
   - 或者访问中文镜像：https://nodejs.org/zh-cn/

2. **下载安装包**
   - 选择 LTS（长期支持）版本，这是推荐的稳定版本
   - 下载 Windows 安装包（.msi 文件）

3. **运行安装程序**
   - 双击下载的 .msi 文件
   - 按照安装向导完成安装（保持默认选项即可）
   - 安装程序会自动将 Node.js 和 npm 添加到系统 PATH 环境变量中

4. **验证安装**
   安装完成后，重新打开命令提示符或 PowerShell，运行以下命令验证：

   ```bash
   node --version
   npm --version
   ```

   如果显示版本号，说明安装成功！

### 方法 2：使用 Chocolatey（如果已安装）

如果您已经安装了 Chocolatey 包管理器，可以使用以下命令：

```powershell
choco install nodejs
```

### 方法 3：使用 winget（Windows 10/11）

```powershell
winget install OpenJS.NodeJS.LTS
```

## 安装完成后

安装完成后，您就可以在项目目录中运行以下命令来安装项目依赖：

```bash
npm install
```

然后启动开发服务器：

```bash
npm run dev
```

## 常见问题

### 如果命令提示符中找不到 node 或 npm

1. 重新打开命令提示符或 PowerShell（需要重新加载环境变量）
2. 如果仍然不行，检查 Node.js 是否已正确添加到系统 PATH 环境变量中
3. 可以尝试重新安装 Node.js，确保在安装时选择"添加到 PATH"选项

### 需要更新 npm

如果 Node.js 已安装但 npm 版本较旧，可以使用以下命令更新：

```bash
npm install -g npm@latest
```

## 推荐版本

- Node.js LTS 版本（推荐用于生产环境）
- 当前推荐版本：Node.js 20.x 或 18.x
