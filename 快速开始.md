# 快速开始指南

## 当前状态

根据终端输出，您的系统上**尚未安装 Node.js 和 npm**。

错误信息：
```
'node' 不是内部或外部命令，也不是可运行的程序或批处理文件。
'npm' 不是内部或外部命令，也不是可运行的程序或批处理文件。
```

这意味着您需要先安装 Node.js（npm 会随 Node.js 一起安装）。

## 🔽 立即安装 Node.js（包含 npm）

### 步骤 1：下载 Node.js

**方式 A：官网下载（推荐）**
1. 打开浏览器，访问：**https://nodejs.org/zh-cn/**
2. 点击绿色的 "**推荐大多数用户使用的版本**"（LTS 版本）
3. 会自动下载 Windows 64 位安装包（例如：node-v20.x.x-x64.msi）

**方式 B：直接下载链接**
- 访问：https://nodejs.org/zh-cn/download/
- 选择 Windows Installer (.msi) 64-bit

### 步骤 2：安装 Node.js

1. 找到下载的 `.msi` 文件（通常在"下载"文件夹中）
2. **双击**运行安装程序
3. 在安装向导中：
   - 点击"下一步"
   - 接受许可协议
   - **保持默认安装路径**（通常是 `C:\Program Files\nodejs\`）
   - **重要**：确保勾选"Automatically install the necessary tools"（自动安装必要工具）
   - 点击"安装"
4. 等待安装完成（通常需要 1-2 分钟）
5. 点击"完成"

### 步骤 3：验证安装

**重要**：安装完成后，必须**重新打开**命令提示符或 PowerShell 窗口！

1. **关闭当前的所有命令提示符/PowerShell 窗口**
2. **重新打开**一个新的命令提示符或 PowerShell
3. 运行以下命令验证：

```bash
node --version
```

您应该看到类似这样的输出：
```
v20.11.0
```

然后运行：

```bash
npm --version
```

您应该看到类似这样的输出：
```
10.2.4
```

如果两个命令都显示版本号，说明安装成功！✅

### 步骤 4：安装项目依赖

验证安装成功后，在项目目录中运行：

```bash
npm install
```

这个命令会安装项目所需的所有依赖包（可能需要几分钟）。

### 步骤 5：启动开发服务器

依赖安装完成后，运行：

```bash
npm run dev
```

您应该看到类似这样的输出：
```
> personal-blog@1.0.0 dev
> next dev

  ▲ Next.js 14.0.0
  - Local:        http://localhost:3000
  - Ready in 2.3s
```

然后打开浏览器，访问 **http://localhost:3000** 即可看到您的博客网站！

## 🆘 常见问题

### Q: 安装后还是提示 'node' 不是内部或外部命令

**解决方案**：
1. **完全关闭**所有命令提示符/PowerShell 窗口
2. **重新打开**一个新的窗口
3. 如果还是不行，尝试重启电脑

### Q: 安装时可以选择哪些选项？

**推荐设置**：
- ✅ 添加到 PATH（应该默认勾选）
- ✅ 自动安装必要工具
- 保持默认安装路径

### Q: 需要安装哪个版本？

- **推荐**：LTS（长期支持）版本，例如 v20.x.x 或 v18.x.x
- **不推荐**：Current（最新）版本，可能不稳定

### Q: 安装需要多长时间？

- 下载：1-5 分钟（取决于网络速度）
- 安装：1-2 分钟

### Q: 如何检查是否已添加到 PATH？

1. 按 `Win + R` 键
2. 输入 `sysdm.cpl` 并按回车
3. 点击"高级"标签
4. 点击"环境变量"
5. 在"系统变量"中找到"Path"
6. 检查是否包含类似 `C:\Program Files\nodejs\` 的路径

- 安装后记得重新打开命令提示符/PowerShell
- 如果遇到权限问题，尝试以管理员身份运行命令提示符
