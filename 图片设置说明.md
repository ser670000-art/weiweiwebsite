## 复制图片到网站

为了让图片在网站上正常显示，需要将 `image.png` 复制到 `public` 目录中。

### 方法 1：使用 Node.js 脚本（推荐）

如果已经安装了 Node.js，运行：

```bash
node copy-image.js
```

### 方法 2：使用批处理文件

双击运行 `复制图片.bat` 文件

### 方法 3：手动复制

1. 确保项目根目录中有 `image.png` 文件
2. 确保 `public` 目录存在（如果不存在，创建它）
3. 将 `image.png` 复制到 `public` 目录中
4. 确保文件路径为：`public/image.png`

### 方法 4：使用命令行

在项目根目录运行：

**Windows (PowerShell/CMD):**
```bash
copy image.png public\image.png
```

**或者：**
```bash
xcopy image.png public\ /Y
```

## 验证

复制完成后，检查以下内容：

1. ✅ `public/image.png` 文件是否存在
2. ✅ 运行 `npm run dev` 启动开发服务器
3. ✅ 打开浏览器访问 http://localhost:3000
4. ✅ 检查导航栏左上角是否显示图片logo

## 图片显示位置

图片会在以下位置显示：
- **导航栏左侧**：作为网站logo，显示在"ww个人主页"文字旁边
- 图片尺寸：40x40 像素
- 样式：圆形（rounded-full）

## 如果需要调整图片大小

如果图片显示太大或太小，可以编辑 `components/Navbar.tsx` 文件，修改 `width` 和 `height` 属性：

```tsx
<Image 
  src="/image.png" 
  alt="ww个人主页" 
  width={40}  // 修改这个值
  height={40} // 修改这个值
  className="rounded-full object-cover"
/>
```
